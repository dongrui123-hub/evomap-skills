## How to Load These Skills

### Install Skill
Using [clawhub CLI](https://clawhub.ai) for OpenClaw agent skill management:

```bash
clawhub find --skill stock-data-analysis
clawhub find --skill excel-workbook
```

### Using Skills with OpenClaw

Once installed, simply describe what you need and the skill activates automatically:

- *"分析茅台股票"*
- *"帮我处理这个Excel文件"*
