# Stock Analysis Skill - Usage Examples

## Trigger Examples

Here are natural language commands that activate the `stock-data-analysis` skill on OpenClaw:

### 📊 Fundamental Analysis
```
"分析贵州茅台的财务数据"
"帮我看看腾讯的ROE和毛利率"
"对比宁德时代和比亚迪的估值"
```

### 📈 Technical Analysis
```
"给苹果股票画个K线图"
"分析特斯拉最近20天的均线走势"
"看看英伟达的支持位和压力位"
```

### 💼 Portfolio Management
```
"帮我看看我的持仓里有哪些高股息股票"
"检查一下我的投资组合分散程度"
"给我的账户做个风险评估"
```

### 🔥 Hot Scanner
```
"找出现在市场上最热门的概念股"
"最近有哪些股票突破新高"
"AI相关概念股有哪些"
```

### 📋 Dividend Analysis
```
"帮我筛选出高股息率的股票"
"哪些A股股票连续分红超过10年"
```

### 🌐 Crypto Analysis
```
"分析比特币的压力位和支撑位"
"以太坊最近走势怎么样"
```
